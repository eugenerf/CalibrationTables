				Справка
Приложение предназначено для конвертирования градуировочных таблиц, предоставляемых отделом метрологии в виде документов MS Word, в текстовые файлы формата:
	<уровень 1> <объем 1>
	.
	.
	.
	<уровень N> <объем N>

		Описание интерфейса и возможностей приложения
Интерфейс организован в виде текстовых меню. Перемещение между пунктами меню осуществляется посредством клавиш-стрелок. Ряд меню содержит выделенные цветом дополнительные меню опций. Переход между основным меню и меню опций осуществляется с помощью клавиши <Tab>.

	Меню "Градуировочные таблицы" (основное)
1. Одна таблица  - Работа с одной градуировочной таблицей.
	Выполняются следующие действия:
		- выбор и открытие документа Word,содержащего градуировочную таблицу,
		- чтение из него таблицы,
		- изменение базы уровней (единиц измерения уровней) в таблице в соответствии с опциями,
		- сохранение текстового файла с преобразованной таблицей,
		- вызов меню "Таблица: ".

2. Много таблиц - Пакетная обработка нескольких градуировочных таблиц.
	Выполняются следующие действия:
		- выбор папки, содержащей документы Word с градуировочными таблицами,
		- открытие каждого файла и чтение из него таблицы,
		- изменение базы уровней (единиц измерения уровней) в таблицах в соответствии с опциями,
		- сохранение текстовых файлов с преобразованными таблицами,
		- вызов меню "Несколько градуировочных таблиц".

3. Справка     - Вывод данной справки.
4. Выход       - Выход из приложения.

	Меню "Градуировочные таблицы" (Опции)
Записывать заголовок Valcom:             - Выбор записывать или нет заголовок Valcom в текстовые файлы. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.

Исправлять непоследовательность объемов: - Выбор исправлять ли в таблицах непоследовательность объемов. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.

База уровней в выходных файлах:          - Задает базу уровней (единицы измерения) в выходных текстовых файлах. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.

Писать log-файл:                         - Выбор писать или нет лог действий приложения в файл. После двоеточия отображено текущее значение настройки. Лог-файл будет расположен рядом с исполняемым файлом приложения.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.


	Меню "Таблица: " (основное, после двоеточия отображается название таблицы)
1. Отобразить таблицу       - Вывод на экран содержимого градуировочной таблицы.
2. Изменить базу уровней на - Изменение базы уровней на указанное в пункте.
3. Правка значений таблицы  - Вызов меню "Правка таблицы ".
4. Проверить таблицу        - Производит проверку таблицы на наличие ошибок.
	Выполняются следующие действия:
		- проверка на отсортированность (для корректной работы в системе управления градуировочная таблица должна быть отсортирована по возрастанию уровней),
		- проверка на последовательность уровней (увеличение уровня всегда идет на одно и то же значение, в зависимости от заданной базы уровней),
		- проверка на последовательность объемов (в отсортированной таблице значения объемов также должны идти по возрастанию).

5. Исправить таблицу        - Исправляет ошибки в градуировочной таблице.
	Выполняются следующие действия:
		- производится сортировка таблицы (при необходимости),
		- устраняется непоследовательность уровней (при необходимости),
		- устраняется непоследовательность объемов (при необходимости).

6. Сортировать              - Градуировочная таблица сортируется по возрастанию уровней.
7. Сохранить                - Градуировочная таблица сохраняется в текстовый файл.
8. Выход                    - Выход из приложения.

	Меню "Таблица: " (Опции)
Исправлять непоследовательность объемов: - Выбор исправлять ли в таблицах непоследовательность объемов. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.

Записывать заголовок Valcom: - Выбор записывать или нет заголовок Valcom в текстовые файлы. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.


	Меню "Несколько градуировочных таблиц"
1. Выбрать таблицу для работы     - Вызов меню "Выберите таблицу для работы"
2. Изменить везде базу уровней на - Изменение базы уровней во всех таблицах на указанное в пункте.
3. Проверить все таблицы          - Производит проверку всех таблиц на наличие ошибок.
4. Исправить все таблицы          - Исправляет ошибки во всех градуировочных таблицах.
5. Сортировать все таблицы        - Все градуировочные таблицы сортируются по возрастанию уровней.
6. Сохранить все таблицы          - Все градуировочные таблицы сохраняются в текстовые файлы.
7. Выход                          - Выход из приложения.

	Меню "Таблица: " (Опции)
Исправлять непоследовательность объемов: - Выбор исправлять ли в таблицах непоследовательность объемов. После двоеточия отображено текущее значение настройки.
	Выполняются следующие действия:
		- вызов подменю, в котором выбирается значение для настройки.


	Меню "Выберите таблицу для работы"
1. Назад - Возврат в меню "Несколько градуировочных таблиц"
2.       - Этот и все следующие пункты называются по имени соответствующих градуировочных таблиц. Вызывает меню "Таблица: ". Отличие меню "Таблица: ", вызванного из данного меню от такого же меню "Таблица: ", описанного выше, лишь в пункте №8. Тут пункт №8 будет иметь название "8. Назад", и будет производить возврат в меню "Выберите таблицу для работы".

	Меню "Правка таблицы " (продолжение названия - имя таблицы)
1. Добавить запись - Добавление записи <уровень>-<объем> в градуировочную таблицу. Если добавляемый уровень уже существует в таблице, то будет произведено изменение объема для данного уровня.
2. Изменить запись - Изменение в таблице объема для заданного уровня. Навигация осуществляется клавишами-стрелками, PgUp, PgDown, Home и End. Также доступен поиск по уровню: при вводе значения уровня оно отображается в поле "Поиск уровня" и автоматически выделяется строка с уровнем, ближайшим к началу таблицы и начинающимся с введенного значения. Если введенный уровень не найден, поле автоматически очищается и выбирается первая строка таблицы. Выход через клавишу <Esc>.
3. Удалить запись  - Удаление из таблицы записи по заданному уровню. Доступны навигация и поиск (аналогично предыдущему пункту). Удаление пункта через его выбор (клавиша <Enter>).
4. Назад           - Возврат в меню "Таблица: "
